【空间位置与毛玻璃背景要求】
1. 这是一个侧边抽屉面板（Side Drawer），固定在应用的最右侧，VerticalAlignment="Stretch"，宽度约为 360px。
2. 背景必须实现“毛玻璃/亚克力（Acrylic/Blur）”效果，具有半透明感，能隐约看到下方被遮挡的底层内容。边缘需要有 1px 的淡灰色描边或极轻的向左阴影以增强层次感。

【动画要求】
1. 入场动画：当面板显示时，从右侧屏幕边缘向左平滑滑出（RenderTransform Canvas.X 或 TranslateTransform），同时不透明度从 0 渐变为 1（Opacity 0 to 1）。耗时 300ms，使用 EaseOut 效果。
2. 退场动画：点击关闭时，向右滑出屏幕并淡出。

【详细布局结构还原（从上往下）】
请在面板内部使用 ScrollViewer 包裹以下垂直堆叠的内容：

1. 头部区域：
   - 左侧为 Mod 的大标题文本（加粗，16px，如 "Special K for Wuthering Waves"）。
   - 右侧为一个精致的关闭按钮（X 图标）。

2. 作者信息与数据栏：
   - 包含：[圆形作者头像] + [作者名] + [发布时间]。
   - 下方水平排列两个统计标签：一个是点赞数（💙 5 赞），一个是浏览数（👁️ 547 浏览）。

3. Tab 切换栏：
   - 类似 Segmented Control 的左右切换菜单，选项为：“总览与文件” 和 “图文描述”。当前选中项下方有蓝色指示线。

4. 媒体画廊区（Gallery）：
   - 一个可以横向滚动的图片区域，展示 Mod 截图。主图圆角包裹，右侧边缘隐约露出下一张图的一部分以引导滑动。

5. 外部链接按钮：
   - 一个全宽的圆角按钮，背景为微显的浅灰色，文字为“在浏览器中查看”。

6. 可下载文件列表（Files Section）：
   - 包含一个局部小标题：“可下载文件”。
   - 下方是具体的文件卡片项目，采用左右布局：
     * 左侧垂直排列：文件名称（加粗，如 "injector.zip"）、说明文本（灰色，小字）、文件属性（如 "240K  14次下载"）。
     * 右侧：一个正方形的坚实蓝色背景按钮（#0078D4），中心为一个白色的下载图标（📥）。

【代码质量要求】
- 如果是 XAML，请给出完整的动画 Storyboard 定义或 Behavior 代码。
- 毛玻璃效果请使用当前技术栈下最标准的半透明/亚克力实现方案（如 Avalonia 的 ExperimentalAcrylicBorder 或 WPF 的 WindowChrome/Blur 方案，并在代码中注明）。
- 图标部分可先用常见 Unicode 字符（❌, 💙, 👁️, 📥）或 Path 路径代替。
